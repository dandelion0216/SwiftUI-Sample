# SwiftUI-Sample
## 概要
このリポジトリは[こちら]()のブログ用に作成したプロジェクトを管理しています。  
`blog_sample.xcworkspace`このワークスペースを開くことで各サンプルプロジェクトを見れます。  

### 各プロジェクトの内容
- Blog-sample-SwiftUI
  - SwiftUIオンリーのプロジェクト
- Blog-sample-storyboard
  - StoryboardでSwiftUIを使っているプロジェクト
- Blog-sample-xib
  - XIBでSwiftUIを使っているプロジェクト

## 利用
クローンするだけでシミュレーターでは実行できるはずです。
